# EIN
### Elektronikus iskolanapló

#### Használt technológiák
* JPA (Hibernate)
* H2 DB (memóriában)
* Spring Boot 2 (Spring MVC, Spring Data JPA)
* JSP
* CSS, HTML

#### Projekt indítása
Fejlesztés: `mvn clean install spring-boot:run`  
WAR: `java -jar target/ein-0.1.0-SNAPSHOT.war`  
Böngésző: `localhost:8080`
